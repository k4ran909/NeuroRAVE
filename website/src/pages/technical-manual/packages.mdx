# Neuro RAVE Packages

The [Neuro RAVE repo](https://github.com/k4ran909/NeuroRAVE) is organized as a monorepo, containing multiple npm packages.
The purpose of the multiple packages is to

- organize the codebase into more modular, encapsulated pieces
- be able to opt out of certain functionalities
- keep the dependencies of the core packages small

## Overview

[See the latest published packages on npm](https://www.npmjs.com/search?q=%40strudel).
Here is an overview of all the packages:

### Umbrella Packages

These packages give you a batteries-included point of getting started, and most likely the thing you'd want to use in your project:

- [repl](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/repl): The Neuro RAVE REPL as a web component.
- [web](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/web): Neuro RAVE library for the browser, without UI.

To find out more about these two, read [Using Neuro RAVE in Your Project](/technical-manual/project-start)

### Essential Packages

These package are the most essential. You might want to use all of those if you're using Neuro RAVE in your project:

- [core](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/core#strudelcore): tidal pattern engine with core primitives
- [mini](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/mini#strudelmini): mini notation parser + core bindings
- [transpiler](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/transpiler#strudeltranspiler): user code transpiler. syntax sugar + highlighting

### Language Extensions

These packages extend the pattern language by specific functions

- [tonal](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/tonal): tonal functions for scales and chords
- [xen](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/xen): microtonal / xenharmonic functions

### Outputs

These packages provide bindings for different ways to output Neuro RAVE patterns:

- [webaudio](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/webaudio#strudelwebaudio): the default webaudio output
- [osc](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/osc#strudelosc): bindings to communicate via OSC
- [midi](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/midi#strudelmidi): webmidi bindings
- [csound](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/csound#strudelcsound): csound bindings
- [soundfonts](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/serial#strudelsoundfonts): Soundfont support
- [serial](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/serial#strudelserial): webserial bindings

### Others

- [embed](https://github.com/k4ran909/NeuroRAVE/tree/main/packages/embed#strudelembed): embeddable REPL web component

### No Longer Maintained

- [react](https://www.npmjs.com/package/@strudel.cycles/react): react hooks and components for Neuro RAVE
- [eval](https://www.npmjs.com/package/@strudel.cycles/eval): old code transpiler
- [tone](https://www.npmjs.com/package/@strudel.cycles/tone): bindings for Tone.js instruments and effects
- [webdirt](https://www.npmjs.com/package/@strudel.cycles/webdirt): webdirt bindings, replaced by webaudio package
- any `@strudel.cycles/*` packages have been renamed to `@strudel/*` since version 0.10.0.
